1.Open VS code in your computer.

2.Install the Remote Development Extension Pack for VS Code IDE.

3.Connect VS Code to the remote machine(Ubuntu)

control + shift + p to search Remote-SSH and choose Remote-SSH:Connect Current Window to Host.

choose Add New SSH host. Next step paste your SSH host and choose Linux.

4.Install the PlatformIO and the Arduino framework

You must install python in Ubuntu for run PlatformIO and select port in PlatformIO setting

5.Create a project using PlatformIO and the Arduino framework.

Click "+ New Project"

Add Project name and Board

6.Build the Arduino sketch and upload it to the target.

Click Explorer and click your file project name then click src and main.cpp to open your file.

Write c/c++ to controle arduino board.

Save and compile code.
